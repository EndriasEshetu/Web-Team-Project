# Installation

Steps to get the project running locally.

1.  Install Node.js (v16+ recommended).
2.  From the repository root, install server dependencies:

```bash
cd server
npm install
```

3.  Install client dependencies and start the dev server:

```bash
cd ../client
npm install
npm run dev
```

4.  Start the server (from repository root):

```bash
cd server
npm start
```

5.  Open the client URL shown by the Vite dev server (usually http://localhost:5173).
